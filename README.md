🎙️ Local Voice Assistant

Голосовой помощник с полным сохранением приватности


📖 О проекте
Local Voice Assistant — это оффлайн голосовой помощник для Windows, который не отправляет ваши данные в облако. Все команды обрабатываются локально с использованием нейросетей Vosk (распознавание речи) и Silero (синтез речи).


✨ Ключевые особенности:
🔒 Приватность
Никаких облачных API — ваши голосовые данные не покидают устройство
⚡ Скорость
Мгновенная реакция на команды (0.01 сек для базовых команд)
🎨 Современный UI
Тёмная тема, неоновые акценты, анимированные элементы
🎯 500+ команд
Управление приложениями, системой, браузером, медиа и не только
🌐 Оффлайн
Работает без интернета после первоначальной настройки

🛠 Технологический стек
  🎤 STT (Speech-to-Text)  →  Vosk (offline model)          
│  🔊 TTS (Text-to-Speech)  →  Silero (neural network)       │
│  🖼️  GUI (Interface)        →  CustomTkinter               │
│  🎵 Audio Processing        →  PyAudio, SoundFile          │
│  ⚙️  System Control         →  subprocess, keyboard        │
    🐍 Language                →  Python 3.10+    
📦 Установка
Шаг 1: Клонирование репозитория

git clone https://github.com/pon46726-debug/LocalVoiceAssistant.git
cd LocalVoiceAssistant

Шаг 2: Создание виртуального окружения

python -m venv venv
venv\Scripts\activate  # Windows

Шаг 3: Установка зависимостей

pip install --upgrade pip
pip install pipwin
pipwin install pyaudio
pip install -r requirements.txt

Шаг 4: Скачивание модели распознавания речи
Модель не включена в репозиторий из-за размера (~50 МБ).
Скачайте: vosk-model-small-ru-0.22.zip
Распакуйте архив
Переименуйте папку в vosk-model-small-ru-0.22
Поместите в папку assets/

Использование
Запустите приложение
Нажмите кнопку "▶️ ЗАПУСТИТЬ"
Произнесите команду (например: "Открой браузер")
Ассистент выполнит команду и голосом подтвердит действие

Добавление своих команд
Откройте src/config.py и добавьте новую команду:

open
    Открыть приложение/сайт
start chrome
key
    Нажатие клавиши
volume up
parser
    Парсинг данных (курс, погода)
usd, weather
reply
    Только текстовый ответ
"Привет!"
